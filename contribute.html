If you want to see positive change in your country, be a part of it. Join us on our journey to transform government and impact lives.

Here’s how you can contribute

Join the Mailing Lists

Our project mailing lists are open to the public. If you’re interested in contributing, send a request through lsf-architecture@googlegroups.com or monitor the project-specific mailing lists on the Projects page.

Write Code

LSF invites anyone (not just committers) to contribute via Github issues. Check out the relevant GitHub repository and documentation, select an issue which requires your skill strengths and send a pull request for review.
